\section{Problem Awareness} \label{sec:problem-awareness}

% \href{https://scholar.google.com}{Google Scholar}\footnote{https://scholar.google.com} and the \href{https://dl.acm.org}{ACM Digital Library}\footnote{https://dl.acm.org} were used as primary search engines, with the following search string: \texttt{("theory of mind" OR "ToM") AND ("generative AI" OR "large language model" OR "LLM")}. As the search yielded a large number of results, the search was limited to publications from the last five years and the first 100 results of each search engine were screened for relevance. The relevant literature was then analysed to identify relevant insights to define meta-requirements for the design principles. The meta-requirements were then used to guide the analysis of the data from a previously conducted study, which is described in detail in chapter \ref{sec:study-design}. The analysis of the data was performed through a qualitative coding process inspired by \textcite{Jussupow2021}.
